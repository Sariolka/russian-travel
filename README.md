## Описание проекта
Проект выполнен в рамках учебы на курсе *__Веб-разработчик__* Яндекс.Практикум. Одностраничный сайт про российский внутренний туризм создан по макету Figma.

* HTML
* CSS
* Файловая структура по схеме Nested БЭМ  
* Flexbox  
* Grid Layout  
* Figma  
* Адаптивная верстка
