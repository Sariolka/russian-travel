# Проект: Путешествие по России

## Описание проекта
Проект выполнен в рамках учебы на курсе *__Веб-разработчик__* Яндекс.Практикум. Одностраничный сайт про российский внутренний туризм создан по макету в Figma. 

## Технологии  
* Файловая структура по схеме Nested БЭМ  
* Flexbox  
* Grid Layout  
* работа с Figma  
* Позиционирование элементов
